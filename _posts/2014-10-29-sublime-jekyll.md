지킬 + sublime text 사용자에게 유용한 [sublime-jekyll] 패키지를 소개.



단축키 :


{% highlight ruby %}
Jekyll: New post => CMD+K, CMD+P
Jekyll: New draft => CMD+K, CMD+F
Jekyll: Insert current date => CMD+K, CMD+D
Jekyll: Insert current datetime => CMD+K, CMD+T
{% endhighlight %}
